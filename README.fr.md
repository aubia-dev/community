## Qu'est-ce qu'Aubia

Aubia est un cockpit desktop qui orchestre vos agents IA de codage (Claude Code, Codex CLI, Mistral CLI, Gemini CLI, Ollama) et les fait se relire mutuellement, pour que vous n'ayez plus à le faire seul un dimanche soir.

- **Cross-review** : chaque sortie d'agent passe automatiquement par un second LLM d'un provider différent.
- **Opinions vers Plan** : N agents travaillent en parallèle sur la même tâche, un synthétiseur extrait un plan unifié que vous éditez et exécutez.
- **BYOK strict** : vos contrats Anthropic, OpenAI, Mistral, Google, Ollama restent les vôtres. Aubia n'est pas un revendeur de tokens.
- **Privacy by design** : analytics sans cookies, données isolées en UE par défaut, politique de confidentialité minimaliste.

Cet espace communautaire est l'endroit pour façonner Aubia avant et pendant la beta. Choisissez la bonne catégorie de discussion :

|---|---|
| Poser une question, obtenir de l'aide, trouver un contournement | [Q&A](https://github.com/aubia-dev/community/discussions/categories/q-a) |
| Suggérer une feature, une intégration, un workflow | [Ideas](https://github.com/aubia-dev/community/discussions/categories/ideas) |
| Partager ce que vous construisez avec Aubia | [Show and tell](https://github.com/aubia-dev/community/discussions/categories/show-and-tell) |
| Discuter de tout autre sujet pertinent | [General](https://github.com/aubia-dev/community/discussions/categories/general) |
| Lire les annonces produit, releases, milestones | [Announcements](https://github.com/aubia-dev/community/discussions/categories/announcements) |

L'équipe répond aux nouvelles discussions sous 24 à 48 heures en jours ouvrés.
